# Encrypt password in python
```
$ git clone https://github.com/daniellx42/encrypt-py.git

$ cd encrypt-py

$ chmod +x hash.py
```
---
## Editing file
```
$ vim hash.py
```
Choose which hash you want to use.

Remove `tuturu` and add new password.

save and exit file with `wq`.
---
## Execute 
```
$ ./hash.py
```
